
# HTTP File Upload Server

Минималистичный HTTP сервер для загрузки файлов на C, соответствующий требованиям курсового проекта.

## Возможности

- ✅ HTTP/1.1 сервер на стандартных библиотеках C
- ✅ Обработка POST запросов через сокеты (Linux)
- ✅ Загрузка файлов из браузера (multipart/form-data)
- ✅ HTTP Basic Authentication
- ✅ Конфигурируемые адрес и порт
- ✅ systemd сервис
- ✅ Обработка ошибок
- ✅ HTML форма для тестирования

## Быстрый старт

### Сборка
```bash
make
```

### Запуск
```bash
./simple_http server.conf
```

### Тестирование
```bash
./test.sh
```

Откройте браузер: `http://localhost:8080`

- Linux
- gcc

## Конфигурация

Редактируйте `server.conf`:
- `address` - IP адрес для прослушивания
- `port` - Порт
- `upload_dir` - Директория для сохранения файлов
- `username` / `password` - Учетные данные для Basic Auth
- `auth_enabled` - Включить/выключить авторизацию
